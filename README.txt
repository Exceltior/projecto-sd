Instruções de uso:
1- Instalar base de dados Oracle Express
2- Correr os ficheiros Cria_Tabelas.sql, Cria_Sequencias.sql e Insere_Dados.sql
4- Ligar base de dados (no IP 192.168.56.120)
5- Instalar e configurar correctamente servidor Tomcat
6- Instalar IntelliJ (ou equivalente) e abrir o Projecto
7- Correr os diferentes projectos e run configurations fornecidos
8- Aceder a localhost:8080
